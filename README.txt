'project_1_code_and_binaries_rgopalk.zip' contains the source code and the binaries for each of the three problems. It has 3 directories for each problem and for the 2 and 3, a jar file is added for running it on Hadoop. A README with individual instructions is present each directory as well.

'problem_3_graphs' contains the plots of 'degree vs no of node' to determine whether they are scale-free or not.

The random generated graphs are not uploaded since each one is over 170MB
